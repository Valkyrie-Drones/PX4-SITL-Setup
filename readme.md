# Setup Instructions

1. Go to the home directory
```shell
    cd ~
```

2. Clone the repository
```shell
    git clone https://github.com/Valkyrie-Drones/PX4-SITL-Setup
```

3. Go to the cloned repository
```shell
    cd PX4-SITL-Setup
```

4. Run the setup script
```shell
    bash ./setup.sh
```

5. Restart WSL


# Run Instructions

1. Go to the PX4-SITL folder
```shell
    cd ~/PX4-SITL
```
