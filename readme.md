# Setup Instructions

1. Go to the home directory
```shell
    cd ~
```

2. Clone the repository
```shell
    git clone https://github.com/Valkyrie-Drones/PX4-SITL-Setup
```

3. Run the setup script
```shell
    bash ./PX4-SITL-Setup/setup.sh
```

4. Restart WSL


# Run Instructions

1. Go to the PX4-SITL folder
```shell
    cd ~/PX4-SITL
```
